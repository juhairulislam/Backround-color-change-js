# Interactive Welcome & Background Switcher

## Overview

This project demonstrates core JavaScript fundamentals through interactive UI behavior.

Users can enter their name to receive a personalized welcome message.
Additionally, 12 interactive buttons dynamically change the background color without reloading the page.

The goal of this project is to strengthen understanding of DOM manipulation and event-driven programming.

# Features

* User input handling

* Dynamic welcome message rendering

* 12 interactive background switch buttons

* Real-time DOM updates



# Technologies Used

* HTML5

* CSS3

* JavaScript 


## Learning Focus

This project was built to practice:

* DOM selection and manipulation

* Event listeners and click handling

* Handling user input from form fields

* Updating UI dynamically using JavaScript
